## 🐳 Docker

### 🚀 Local Run the landing-page

```
docker run -p 3000:80 -d --rm --name landing-page fairhivelabs/landing-page
```

You can now reach the landing-page using the URL http://localhost:3000

### 👋 Stop and Clean

```
docker stop landing-page
```
